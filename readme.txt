Multi-vendor marketplace plugin for WooCommerce with subscriptions, AI, commissions, and WhatsApp.

== Description ==

Vendor Marketplace transforms your WooCommerce store into a fully-featured multi-vendor marketplace.

= Features =
* Vendor registration and management
* Product management per vendor
* Commission system
* Withdrawal system
* Subscription plans (Free, Pro, Business)
* AI-powered product generation
* WhatsApp integration
* REST API support
* Advanced analytics and reporting

= Requirements =
* WordPress 5.8+
* WooCommerce 6.0+
* PHP 7.4+

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/vendor-marketplace`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure WooCommerce is installed and activated
4. Go to Vendor Marketplace > Settings to configure the plugin
